# Train Schedule and Reservation Management System
 
## Project Overview
A unified platform designed to provide efficient scheduling and reservation services for trains. The system enables management of passenger bookings and ticketing processes while centralizing data for staff and passengers.

## Project Objectives
* Provide a secure user interface for booking and viewing train details.
* Increase data accuracy and reduce human error in bookings.
* Implement Role-Based Access Control (RBAC) to ensure system security.
* Support decision-making through reporting and analytics tools.

  1. Run `npm install` to install dependencies.
  2. Run `npm run dev` to start the local server.
